Barebones template to get started with Solidity projects.

## Install

```sh
git clone https://github.com/0xmichalis/sol-template.git
cd sol-template
forge install
```

## Build

```sh
forge build
```

## Test

```sh
forge test
```

## Update Gas Snapshot

```sh
forge snapshot
```
